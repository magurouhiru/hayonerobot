# ui-primitives

このパッケージは、Dioxusの公式コンポーネントや、汎用的なUIプリミティブを格納するための場所です。

## 役割

- **汎用性**: アプリケーションのビジネスロジックに依存しない、再利用可能なコンポーネントを配置します。
- **公式コンポーネント**: Dioxusが提供する、あるいはそれに準ずる基本的なUI部品（ボタン、入力フィールド、カードなど）をここに実装・管理します。

## ui パッケージとの違い

- `packages/ui`: アプリケーション固有のコンポーネント（特定のビジネスロジックを含むものや、アプリ専用のデザインが強く適用されたもの）を格納します。
- **推奨**: 可能な限り `ui-primitives` にあるコンポーネントを使用し、アプリ固有の要件がある場合のみ `ui` パッケージでラップしたり、独自のコンポーネントを作成したりしてください。
